# MyneChat

MyneChat is a web application developed by [HOPR Association](https://hoprnet.org/), powered by a [HOPR](https://github.com/hoprnet/hoprnet) node. The following guide showcases MyneChat application and its most important components. The application was developed using [Next.js](https://nextjs.org/), [Grommet](https://v2.grommet.io/) and [Immer.js](https://github.com/immerjs/use-immer).

Follow the respective instructions below to have further understanding of the project, and how you can tweak it to develop a HOPR-powered chat for yourself. All code is open source under an MIT License, and can be found [here](https://github.com/hoprnet/myne-chat).
